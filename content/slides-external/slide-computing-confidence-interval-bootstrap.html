Computing confidence intervals

The distributions of sample statistics considered so far require us to either already know the value of the population parameter or to have the resources to take thousands of different samples. In most situations, neither of these is an option.
We usually only have access to the one sample we collected to make inferences about the population. How could we estimate the parameter we are interested in and compute the confidence intervals just from this unique sample?
Think about it:
The sample is the only information we know to be true about the population it has been drawn from.
The best estimate for a population parameter (e.g.: $\pi$) is the relevant sample statistic (e.g.: $\hat{p}$).
We can expand this idea and consider the sample as our best estimate for the underlying population.

We assume that the population of all the data we have is basically just many, many copies of the same data in our original sample

Drawing a bootstrap sample

In practice, instead of actually making many copies of the sample and sampling from that, we use a sampling technique that is equivalent:
we sample with replacement from the original sample.

Once a data value has been selected from the sample, it is still available to be selected again.
Each data value in the original sample actually represents many other data with a similar value. Each sample selected in this way, with replacement from the original sample, is called a bootstrap sample.

Note: Recall that the variability of a sample statistic depends on the size of the sample. Because we are trying to uncover the variability of the sample statistic, it is important that each bootstrap sample is the same size as the original sample.

Generating a Bootstrap distribution

  1. Generate bootstrap samples by sampling with replacement
    from the original sample, using the same sample size
  2. Compute the statistic of interest ("bootstrap statistic")
    for each of the bootstrap samples
  3. Collect the statistics for many bootstrap samples
    to create a bootstrap distribution
