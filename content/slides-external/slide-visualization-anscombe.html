Why visualize your data?

But there are in fact 4 distinct stories here.
And you would see that some mistakes have been made (linear fit for non-linear data, etc ...)
So visualization is a pre-required set to guide you to know which kind of data analysis you can apply, which kind of statistical set you'll be able to apply.
So it is important to know how to describe and represent data (transition).
