2 Simulation
What about we just simulate the experiment? 

            import numpy as np

            M=0 # variable to store positive results

            for i in range(10000):
              # draw 28 numbers between 0 and 1
              trial = np.random.rand(28)<0.5
              # count how many heads we got
              nHeads = np.sum(trial)
              # if more than 19 heads, update M
              if nHeads >= 19:
                M += 1

            p = M/10000 #0.0439
