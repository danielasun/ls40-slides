Calculating correlation coefficients

Pearson's correlation coefficient 
        
          # import Scipy
          from scipy import stats

          stats.pearsonr(x, y)
          #(0.7113662380943626, 0.00043665572481862763)
Spearman's correlation coefficient 
        
          # import Scipy
          from scipy import stats

          stats.spearmanr(x, y)
          # SpearmanrResult(correlation=0.7596374052071106, pvalue=0.00010221194576957504)
