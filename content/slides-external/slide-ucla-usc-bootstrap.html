Estimating $\pi_{\textrm{win}}$ for UCLA against USC

Let's get back to the UCLA-USC basketball games investigation

What is the true probability of UCLA to win a game against USC?
In other words, if UCLA and USC were to play an infinite number of basketball games, what parameter $\pi_{\textrm{win}}$ would we observe on the long-run?

We can use the bootstrap technic to compute the 95% confidence interval of $\pi_{\textrm{win}}$ for UCLA:
  • Reapeatedly randomly draw with replacement 249 UCLA-USC games outcomes from the sample we have and calculate each time the value of our bootstrap statistic $\hat{p}$.
  • Once a UCLA-USC game has been selected from the sample, it is still available to be selected again (drawing with replacement).
    Each UCLA-USC game in the original sample actually represents many other games with a similar outcome (win or loss).

Estimating $\pi_{\textrm{win}}$ for UCLA against USC

Generating a Bootstrap distribution

data sample (n=249)
140 ones & 109 zeros
$P(1)=0.562$
$P(0)=0.438$

1 1 0 0 0 1 ... 1 0 1 1 0 1

$\hat{p}=0.562$

Bootstrap confidence intervals

Now that we have our Bootstrap sampling distribution, we can compute the confidence intervals at the 95% level.

Where is it centered?
centered at $\hat{p}$
(original observed proportion)

How do we compute the 95% confidence intervals of our population parameter?
We consider the central 95% of the bootstrap statistics (discard the 2.5% more extreme cases on each side)

We are 95% confident that the true probability of the Bruins to win a game against the Trojans (parameter $\pi_{\textrm{win}}$) is lying in the interval $[0.498-0.622]$

Reporting confidence intervals instead of p-values

Confidence intervals provide information about both:

  • strength of evidence (Is the parameter value we want to test against, in the interval we obtained?)
  • uncertainty/estimate of the true parameter value (for a chosen $\alpha$ as confidence level, $\alpha$% of the time our interval will include our true parameter)

In that sense, reporting the $\alpha$% confidence intervals is better than just reporting a p-value, since what we really want to know is the true value of our population parameter, not just if our true parameter is different or not from a particular value.

Versatility of the Bootstrap

  • The bootstrap procedure does not require any assumption about the distribution of your data
    (the data sample is the best approximation of the population)
  • The bootstrap procedure can be applied to any data type
    (categorical or quantitative variables)
  • The bootstrap statistic can be anything you're interesed in
    ($\hat{p}$, median, mean, ratio, correlation coefficient, etc ...)
  • The bootstrap procedure can be adapted to replace any theoretical statistical test
    (confidence intervals generation, comparison of groups, etc ...)
