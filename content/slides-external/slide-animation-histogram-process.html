From dotplot to histogram

A dotplot can be hard to build, and dots can overlap (visually) if the values are very similar and a lot of values are represented. Building a histogram is a great way to summarize a variable. Histograms use counts to aggregate similar values together and show the overall distribution of the data.
Processus to construct a histogram
1 Define "boundaries" (they form bins)
2 Count the number of elements inside each bin
A histogram lets you discover the underlying frequency distribution (shape) of a set of data.
